# python 面向对象(第六周)

## 学习任务

1. 学习python的面向对象编程，掌握以下核心概念

    1. 类与对象（如何定义类、如何创建对象...）
    2. 属性与方法（实例属性和类属性；实例方法、类方法和静态方法；self的含义...）
    3. 构造方法（用处、默认参数和可选参数...）
    4. 封装（私有属性与方法的约定...）
    5. 继承（子类与分类、方法重写、super()的使用...）
    6. 多态
    7. 魔法方法
    8. 抽象基类与接口（可选）

2. 练习1：学生管理系统、银行账户模拟、图书馆管理系统等，选择一个或自己想一个，使用类来封装系统中出现的类和方法。

3. 练习2：将下面的项目main.py的代码使用类进行封装，最终实现的效果是其他人能通过你写的tool.py（假设你封装的类所在的文件）调用类的方法完成功能。项目的功能是给定一个视频的路径，使用大模型处理，返回这个视频的开始时间和结束时间及视频的持续时间。
