# 第七周（数据库/MySQL）

## 学习任务
- 了解数据库的基本概念（数据库、表、字段、行、列等）
- 安装并配置MySQL数据库
- 学习使用MySQL的基本命令（如创建数据库、创建表、插入数据、查询数据等）
- 了解Python如何连接和操作MySQL数据库（使用`mysql-connector-python`库）
- 练习：设计一个简单的学生信息管理系统，使用MySQL存储
    - 学生表（字段：学号、姓名、年龄、性别、班级等）
    - **该部分不强求使用python代码，只需要知道如何使用MySQL的基本命令即可**
- 实现功能：
    - 添加学生、班级和教师信息
    - 查询学生信息（按学号、姓名等条件查询）
    - 更新学生信息
    - 删除学生信息
    - **该部分使用python实现**
- 重点：在代码中对于一些操作(增删改查)，了解此时应该给数据库传递哪些参数（如连接数据库需要提供ip、端口等）即可
