# python 基础学习(第五周)

## 学习内容

1. 模块：基本概念，不同的导入方式；常见的标准库(os,time等)，第三方模块管理工具pip，虚拟环境(选学，目前主流使用conda，会用conda可以不学，如果学习可以了解一下[uv](https://uv.oaix.tech/),比conda速度快很多，了解模块设计的原则(以后写自己的模块时尽可能遵守)。

2. 文件操作：读取、写入、追加文件(默认以txt文本文件练习即可)，使用with关键词简化文件操作步骤。

3. 练习：创建一个txt文本文件，内容格式为：每一行包括两个元素，元素之间使用空格隔开，第一个元素为用户名，第二个元素为密码。
结合原先的登录功能，将文件内容读取，当用户输入时，根据读取的内容来判断，是否登录成功。再写一个注册功能，结合之前的用户名和密码有效验证代码，用户输入的账号密码符合要求后将新账号追加到该文件中。将功能(如用户名验证)以自定义模块的方式导入并使用。选作：将该功能集成为一个main.py中，命令行中提供界面供用户选择，如用户输入1则进入登录功能，提示用户输入相应数据后打印结果，此时可以让用户输入三次也可直接返回到最初的功能选择界面。用户输入2进入注册功能，完成注册相关功能后返回最初的功能选择界面。
```
user1 password1
user2 password2
```
