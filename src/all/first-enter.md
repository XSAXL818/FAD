# 初次了解（第一周10月9日-10月10日）

本周学习目标：

1. 掌握如何管理开发工具和项目代码的目录

D盘下创建文件夹Development_Tools和文件夹Code,前者专门放用于开发的软件,后者专门放项目的代码,
